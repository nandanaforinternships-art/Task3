# Linear Regression - AI & ML Internship Task 3

## Project Overview

This project is part of the AI & ML Internship Task 3. The objective is to implement and understand **Simple and Multiple Linear Regression** using Python and Scikit-learn.

The model predicts a dependent variable (e.g., house price) based on one or more independent variables (e.g., area, number of bedrooms).

---

## Objectives

* Implement **Simple Linear Regression**
* Implement **Multiple Linear Regression**
* Evaluate model performance using:

  * MAE (Mean Absolute Error)
  * MSE (Mean Squared Error)
  * R² Score
* Visualize regression results
* Interpret model coefficients

---

## Model Evaluation Metrics

* **MAE (Mean Absolute Error)**
  Measures average absolute difference between predicted and actual values.

* **MSE (Mean Squared Error)**
  Penalizes larger errors more than MAE.

* **R² Score**
  Represents how well the model explains the variance in data.
